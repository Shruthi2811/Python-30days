**A beginner-friendly 30-day practice repo for brushing up on Python data manipulation for interviews.**
This repository is designed for learners who want to build confidence from the basics using a hands-on approach. The focus is on solving small, practical problems with Python first, and then gradually moving into tabular data work with pandas.

**Who This Is For**
This repo is a good fit if you:
* are preparing for Python interviews as a beginner
* want to practice data manipulation step by step
* learn best by writing and running code
* want a structured plan instead of random interview questions


**What You Will Practice**
Over 30 days, you will work through:

* list manipulation
* string processing
* dictionary operations
* counting and grouping patterns
* sorting and filtering
* nested data structures
* CSV and JSON handling
* introductory pandas
* joins, grouping, missing values, and reshaping
* Learning Approach


**30-Day Plan Overview
Week 1: Python Basics For Data Manipulation**
* lists
* strings
* dictionaries
* loops and conditions
* simple counting and filtering

**Week 2: Intermediate Core Problems**
* sorting
* chunking
* finding pairs
* flattening nested structures
* grouping list-of-dictionary data

**Week 3: Files and Intro to Pandas**
* text file handling
* CSV parsing
* JSON parsing
* DataFrame creation
* filtering and null handling in pandas

**Week 4: Cleaning, Grouping, and Interview Practice**
* creating derived columns
* groupby
* sorting and deduplication
* joins
* reshaping data
* text cleaning
* mock interview practice


**Topics You Should Know By The End
By the end of this 30-day plan, you should be comfortable with:**

* list comprehensions
* dictionary counting with get()
* sets for uniqueness and membership checks
* sorting with key
* working with nested data
* reading CSV and JSON
* basic pandas filtering
* handling missing values
* grouping and aggregation
* joins and duplicates
